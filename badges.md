## 使用说明

1. 复制需要的徽章代码
2. 替换 `your-username` 为您的GitHub用户名
3. 根据需要调整徽章样式和颜色
4. 在README.md或其他文档中使用

## 自定义徽章

您可以使用 [Shields.io](https://shields.io/) 创建自定义徽章：

```
https://img.shields.io/badge/标签-值-颜色?style=样式&logo=图标
```

## 徽章样式

- `style=for-the-badge`: 大尺寸徽章
- `style=flat`: 扁平化徽章
- `style=flat-square`: 扁平化方形徽章
- `style=plastic`: 塑料质感徽章
