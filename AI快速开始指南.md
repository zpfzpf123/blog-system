# 🚀 AI快速开始指南

## 5分钟快速配置

### 方案1: 零配置开始 (推荐新手)

1. 直接使用"本地规则生成"
2. 无需任何配置，立即可用
3. 适合快速测试和学习

### 方案2: Hugging Face (推荐个人用户)

1. 访问 [Hugging Face](https://huggingface.co/settings/tokens)
2. 注册账号，创建API密钥
3. 在博客系统中选择"Hugging Face"
4. 输入API密钥，选择模型
5. 测试连接，保存配置

### 方案3: Ollama本地部署 (推荐隐私要求高)

1. 运行安装脚本：
   - Windows: 双击 `install-ollama.bat`
   - Linux/Mac: 运行 `./install-ollama.sh`
2. 在博客系统中选择"Ollama (本地)"
3. 服务地址: `http://localhost:11434`
4. 模型名称: `qwen` 或 `llama2`

## 🎯 推荐配置

### 个人博客用户

- **服务**: Hugging Face
- **模型**: `microsoft/DialoGPT-medium`
- **优势**: 免费额度大，中文友好

### 企业/团队用户

- **服务**: OpenAI 或 Claude
- **模型**: `gpt-3.5-turbo` 或 `claude-3-haiku`
- **优势**: 质量高，稳定性好

### 隐私敏感用户

- **服务**: Ollama本地部署
- **模型**: `qwen` 或 `chatglm`
- **优势**: 完全本地，数据不外传

## ⚡ 快速测试

1. 创建新博客文章
2. 输入标题和内容
3. 选择摘要生成策略
4. 点击"AI生成摘要"
5. 查看生成结果

## 🔧 常见问题

**Q: 为什么AI生成失败？**
A: 检查网络连接、API密钥、服务状态

**Q: 如何提高摘要质量？**
A: 选择更合适的模型、调整生成策略

**Q: 免费额度用完怎么办？**
A: 切换到本地规则生成或Ollama
