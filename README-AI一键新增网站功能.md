# AI一键新增网站功能使用说明

## 功能概述

AI一键新增网站功能允许用户通过输入网站地址，自动获取网站信息并生成网站描述、分类建议等，同时支持用户对AI生成的内容进行手动修改和编辑。

## 主要特性

### 🚀 智能生成

- **自动抓取**：输入网站地址后自动抓取网站基本信息
- **AI分析**：使用AI分析网站内容，生成合适的名称和描述
- **分类推荐**：智能推荐最合适的分类

### ✏️ 手动编辑

- **完全可编辑**：AI生成的所有内容都支持手动修改
- **实时预览**：编辑时实时显示网站图标预览
- **多分类支持**：支持为网站选择多个分类
- **表单验证**：确保所有必填字段都已填写

### 🎨 用户友好

- **分步操作**：清晰的两步操作流程
- **视觉反馈**：美观的界面设计和状态提示
- **错误处理**：完善的错误提示和异常处理

## 使用流程

### 第一步：输入网站地址

1. 点击"AI一键新增网站"按钮
2. 在对话框中输入网站地址（如：https://github.com）
3. 点击"AI生成"按钮

### 第二步：编辑网站信息

1. AI生成完成后，系统会自动填充网站信息
2. 用户可以修改以下字段：
   - **网站名称**：最多50个字符
   - **网站描述**：最多200个字符
   - **所属分类**：支持选择多个分类
   - **网站图标**：显示抓取到的favicon（只读）

3. 编辑完成后点击"保存网站"

## 功能详解

### AI生成内容

AI会根据网站内容自动生成：

- **网站名称**：基于网站标题和内容分析
- **网站描述**：总结网站的主要功能和特点
- **分类推荐**：根据网站内容推荐最合适的分类

### 手动编辑功能

用户可以对AI生成的内容进行以下修改：

#### 网站名称编辑

- 支持中英文输入
- 字符限制：最多50个字符
- 实时字符计数显示
- 必填字段验证

#### 网站描述编辑

- 支持多行文本输入
- 字符限制：最多200个字符
- 实时字符计数显示
- 必填字段验证

#### 分类选择

- 支持多选分类
- 显示分类颜色标识
- 支持折叠标签显示
- 必填字段验证

#### 图标预览

- 显示抓取到的网站favicon
- 加载失败时显示默认图标
- 只读显示，不可编辑

### 重新生成功能

- 如果对AI生成的结果不满意，可以点击"重新生成"
- 重新生成会保持相同的网站地址
- 每次生成可能产生不同的结果

## 技术实现

### 前端技术

- **Vue 3**：响应式框架
- **Element Plus**：UI组件库
- **TypeScript**：类型安全
- **响应式设计**：适配各种屏幕尺寸

### 后端集成

- **网站抓取**：自动获取网站基本信息
- **AI服务**：调用Ollama进行内容分析
- **数据存储**：保存到数据库

### 数据流程

1. 用户输入网站地址
2. 前端调用后端API抓取网站信息
3. 调用AI服务分析网站内容
4. 生成网站信息并返回前端
5. 用户编辑信息
6. 保存到数据库

## 错误处理

### 常见错误及解决方案

#### 网站地址无效

- **错误提示**：请输入有效的网站地址
- **解决方案**：确保输入完整的URL，包含http://或https://

#### 网站无法访问

- **错误提示**：无法抓取网站信息
- **解决方案**：检查网站是否可访问，或稍后重试

#### AI生成失败

- **错误提示**：AI生成失败，请重试
- **解决方案**：检查网络连接，或稍后重试

#### 保存失败

- **错误提示**：保存网站失败
- **解决方案**：检查必填字段是否完整，或稍后重试

## 最佳实践

- 编写简洁明了的描述

### 分类管理

- 定期整理和更新分类
- 为网站选择合适的分类
