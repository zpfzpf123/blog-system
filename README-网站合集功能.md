# 🌐 网站合集功能说明

## 功能概述

网站合集是一个用于收集、整理和管理个人收藏网站的功能模块。用户可以按照预设分类对网站进行分组展示，支持添加、编辑、删除网站，以及创建自定义分类。

## ✨ 主要特性

### 1. 📁 分类管理

- **预设分类**：开发工具、娱乐休闲、学习资源等
- **自定义分类**：支持创建新的分类，设置分类名称、描述和颜色
- **分类统计**：每个分类显示包含的网站数量

### 2. 🔍 智能搜索

- **多字段搜索**：支持按网站名称、描述、URL和标签搜索
- **实时筛选**：输入关键词后实时显示搜索结果
- **组合筛选**：分类筛选和关键词搜索可以组合使用

### 3. 📱 响应式设计

- **桌面端**：网格布局，多列显示
- **移动端**：单列布局，触摸友好
- **自适应**：自动适应不同屏幕尺寸

### 4. ⚡ 快速访问

- **一键访问**：点击网站卡片直接在新标签页打开网站
- **访问统计**：记录每个网站的访问次数
- **收藏标记**：支持标记特别喜欢的网站

## 🚀 使用方法

### 访问页面

1. 在导航菜单中点击"网站合集"
2. 或直接访问 `/websites` 路径

### 添加网站

1. 点击页面右上角的"添加网站"按钮
2. 填写网站信息：
   - 网站名称（必填）
   - 网站URL（必填，需符合URL格式）
   - 网站描述（必填）
   - 所属分类（必填）
   - 网站图标（可选，图标URL）
   - 标签（可选，可创建新标签）
3. 点击"保存"按钮

### 添加分类

1. 点击页面右上角的"添加分类"按钮
2. 填写分类信息：
   - 分类名称（必填）
   - 分类描述（必填）
   - 分类颜色（必填，使用颜色选择器）
3. 点击"保存"按钮

### 管理网站

- **编辑网站**：点击网站卡片右上角的编辑按钮
- **删除网站**：点击网站卡片右上角的删除按钮
- **访问网站**：直接点击网站卡片

### 筛选和搜索

- **分类筛选**：使用页面中部的分类按钮进行筛选
- **关键词搜索**：使用页面中部的搜索框输入关键词

- **Vue 3**：使用 Composition API 和 `<script setup>` 语法

- **CSS Grid**：响应式网格布局

## 🔧 自定义配置

### 修改预设数据

在 `WebsiteCollection.vue` 文件的 `loadWebsites` 方法中修改 `mockWebsites` 和 `mockCategories` 数组。

### 调整分页设置

修改 `pageSize` 的默认值和分页选项：

```typescript
const pageSize = ref(12) // 默认每页显示12个网站
:page-sizes="[12, 24, 48, 96]" // 分页大小选项
```

### 修改网格布局

在CSS中调整网格列数：

```css
.websites-grid {
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
}
```

## 🔍 搜索算法

### 搜索逻辑

1. **分类筛选**：首先按选中的分类筛选网站
2. **关键词搜索**：在筛选结果中搜索关键词
3. **多字段匹配**：搜索网站名称、描述、URL和标签
4. **不区分大小写**：自动转换为小写进行匹配

### 性能优化

- 使用计算属性缓存筛选结果
- 防抖处理搜索输入
- 分页显示，避免一次性渲染大量数据

## 🐛 常见问题

### Q: 网站图标不显示怎么办？

A: 检查图标URL是否正确，如果图标加载失败会自动隐藏并显示默认图标。

### Q: 如何批量管理网站？

A: 目前支持单个操作，后续版本会添加批量操作功能。

### Q: 数据会丢失吗？

A: 当前版本使用模拟数据，刷新页面后会重置。建议后续集成后端API进行数据持久化。

### Q: 支持哪些浏览器？

A: 支持所有现代浏览器，包括Chrome、Firefox、Safari、Edge等。
